# Training Planner - План тренировок
Ссылка на репозиторий:https://github.com/morylevdanila-art/Training-Planner
Автор:Данила Морылев 

Описание

Training Planner - это удобное desktop-приложение для планирования и учёта тренировок. Приложение позволяет:

- ✅ Добавлять тренировки с указанием даты, типа и длительности
- 🔍 Фильтровать тренировки по типу и дате
- 💾 Автоматически сохранять данные в JSON-файл
- 🗑 Удалять записи двойным кликом
- 📊 Отслеживать общую статистику (количество тренировок и общую длительность)

 🚀 Установка и запуск

Требования
- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, os, datetime)

Запуск
```bash
python training_planner.py
